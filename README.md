# Space Gate Studio (Next.js + TypeScript + Tailwind + Motion + 3D + Sound)

## Quickstart
```bash
npm install
npm run dev
```

## Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion (page transitions + micro-interactions)
- GSAP (scroll-triggered timeline animations)
- Three.js / React Three Fiber (space visuals)
- Howler.js (UI sounds with mute toggle)
- Lenis (smooth scrolling)
- Shadcn-style UI primitives (Button, Card)

## Notes
- Sounds are lightweight generated wav files in `/public/sounds`.
- Replace `mailto:hello@spacegatestudio.com` in CTA with your real contact endpoint.
